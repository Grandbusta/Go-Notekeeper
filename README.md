# NoteKeeper API - Go🔥

A Note keeper built using Golang.
More features to come.

Star⭐ the repo if you like what you see😉.

## Technologies Used

- Golang
- JWT
- Gorilla Mux
- GORM
- MySQL

## Features

- Get all notes
- Get a single note
- Create a new note
- Update a note
- Delete note
- User Login and SignUp
